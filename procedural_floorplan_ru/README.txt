
Установка:
1. Blender -> Edit -> Preferences -> Add-ons -> Install...
2. Выбери этот ZIP целиком, не распаковывая.
3. Включи аддон "Процедурный генератор планировки дома".
4. Панель появится в 3D Viewport -> Sidebar -> вкладка "План дома".

Что внутри:
- core.py — логика генератора
- props.py — все настройки аддона
- ui.py — интерфейс панели
- operators.py — кнопки генерации и сброса

Примечание:
Текущая версия собрана на основе patched_v3 скрипта с исправленным уличным проёмом для входной двери.
